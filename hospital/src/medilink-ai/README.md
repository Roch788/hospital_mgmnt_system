# MediSync AI

## Overview
MediSync AI is an innovative healthcare platform designed to enhance patient experience and streamline healthcare services through the use of artificial intelligence. This project serves as a landing page for the MediSync AI platform, showcasing its features and functionalities.

## Features
- **Floating Glass Navbar**: A modern navigation bar with a glassmorphism effect for a sleek look.
- **Hero Section**: Engaging introduction to the MediSync AI platform with a call to action.
- **Live Healthcare Stats**: Real-time statistics showcasing healthcare data.
- **Smart Resource Search**: An intelligent search feature for healthcare resources.
- **Live Hospital Map Preview**: Interactive map displaying hospital locations and services.
- **Platform Features Section**: Detailed descriptions of the platform's capabilities.
- **How It Works Section**: An explanation of the technology and processes behind MediSync AI.
- **Testimonial Section**: User feedback and success stories.
- **Call to Action Section**: Encouraging users to sign up or learn more.
- **Footer**: Contact information and links to social media.

## Technologies Used
- **HTML**: For the structure of the landing page.
- **CSS**: Custom styles using Tailwind CSS for responsive design and aesthetics.
- **JavaScript**: For interactivity and dynamic content.
- **Tailwind CSS**: A utility-first CSS framework for rapid UI development.

## Installation
1. Clone the repository:
   ```
   git clone https://github.com/yourusername/MediSync-ai.git
   ```
2. Navigate to the project directory:
   ```
   cd MediSync-ai
   ```
3. Install dependencies:
   ```
   npm install
   ```
4. Build the project:
   ```
   npm run build
   ```
5. Serve the project:
   ```
   npm start
   ```
